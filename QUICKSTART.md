# Быстрый старт веб-приложения

## Установка и запуск

### 1. Перейти в директорию web

```bash
cd web
```

### 2. Создать виртуальное окружение

```bash
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Применить миграции

```bash
python manage.py migrate
```

### 5. Создать суперпользователя (опционально)

```bash
python manage.py createsuperuser
```

### 6. Запустить сервер

```bash
python manage.py runserver
```

### 7. Открыть в браузере

http://127.0.0.1:8000/

## Первое использование

1. Зарегистрируйте новый аккаунт
2. Войдите в систему
3. Загрузите аудио файл
4. Выберите модель Whisper
5. Нажмите "Распознать речь"
6. Дождитесь завершения обработки
7. Просмотрите транскрипцию

## Настройка фоновых задач (опционально)

Для обработки записей в фоне установите Celery:

```bash
pip install celery redis
```

Запустите worker:

```bash
celery -A voice_recorder worker --loglevel=info
```

Без Celery обработка будет выполняться синхронно (может блокировать запросы).

## Структура

- `/dashboard/` - Главная страница с дашбордом
- `/recordings/` - Список всех записей
- `/recordings/<id>/` - Детали записи
- `/settings/` - Настройки пользователя

## Решение проблем

### Ошибка при миграциях

```bash
python manage.py makemigrations
python manage.py migrate
```

### Ошибка с медиа файлами

Убедитесь что директория `media/` существует и доступна для записи.

### Медленная обработка

- Используйте меньшую модель Whisper (tiny/base)
- Настройте Celery для фоновой обработки
- Используйте более мощный сервер
